# Examples

Reference artifacts that show `tokens.json` in use. Each file is a minimal, self-contained demonstration — not a template to copy wholesale, but a proof that the tokens resolve to a sensible, on-brand result.

When adding an example:

- Keep it short (50–150 lines).
- Reference tokens by name in a comment header, then resolve to literal values in the actual output.
- Cover one artifact type per file. If you need two, add two files.
