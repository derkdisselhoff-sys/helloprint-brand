## Print vocabulary

Use the customer-friendly term unless writing for a professional buyer. When both appear, put the buyer term in parentheses.

| Customer-friendly | Technical | When to use the technical term |
| --- | --- | --- |
| Paper weight | GSM / g/m² | Product spec page, supplier docs |
| Shiny finish | Gloss lamination | Supplier-facing or detailed product spec |
| Matte finish | Matt lamination | Same |
| Card | Business card | Everywhere customer-facing |
| Flyer | Leaflet | NL: "flyer" is fine, UK: both work |
| Box | Corrugated / folding carton | Packaging-specific technical docs |
| Cut-out shape | Die-cut | Supplier docs, pre-press guides |
| File check | Pre-press check | Supplier docs, explainer pages |

## Time and units

- Use `48h` / `72h` over "two days" / "three days" when the SLA is specific.
- Use `€` before the number with no space: `€12,50` (NL/BE) / `€12.50` (UK/IE). Stick to the market's decimal convention.
- Dates in prose: `23 April 2026` (NL/UK). Never `4/23/26`.
- Times: 24-hour clock: `14:00 CET`. Include timezone for anything cross-market.

## Measurements

- Digital: `px` for typography and component sizing.
- Print: `mm` primary, `cm` for large-format, `inches` only if the customer context is explicitly US.

## Inclusive language

- Use "they / them" as singular default when a person's pronouns aren't known.
- Prefer "team" over "guys."
- Prefer "staff / folks / everyone" over gendered plurals.
- Avoid "crazy", "insane", "blind spot" in professional writing — swap for specific alternatives.
